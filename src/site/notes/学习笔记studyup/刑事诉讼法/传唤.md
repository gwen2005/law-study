# 与拘传对比
## 不同
- 性质不同
	- 传唤：属于任意性到案措施，非强制性到案措施
		- 没有必然的到案义务
		- 针对对象更广
	- 拘传：强制性到案措施
- 传唤不是拘传的充分、必要条件

# 形式
1. 书面传唤：不需要拘留、逮捕的嫌疑人
2. 口头传唤：现场发现的嫌疑人
