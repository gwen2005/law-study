# 范围
- 被代理人：未成年人、无行为能力或限制行为能力人
- 代理人：父母、养父母、监护人、负有保护责任的机关和团体代表
# 代理权效果
- 代理权：被代理人的法定诉讼权利
- 代理效果：被代理人承担
