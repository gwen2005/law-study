- 双方代理是指代理人同时代理双方当事人实施法律行为 #背诵/概念 
# 例外
- 代理人同意或者追认的情况下，需要被代理人双方都表示同意或者追认
{ #4b44fe}

- 纯获利的法律行为：法律行为相对于一方被代理人而言纯获利益，相对于另一方被代理人而言并非纯获利益，所以需要另一方被代理人对双方代理表示同意或者追认
{ #8e067b}
