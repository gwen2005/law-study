# 对比：无罪推定和推定无罪
- 无罪推定和事实无关，不是证据法的推定
	- 证据法的推定包括事实推定和法律推定，其目的是形成事实
- 无罪推定属于地位推定，不是结果推定
	- 推定无罪是结果推定
- 无罪推定必然要求的两个制度：控方举证和沉默权
	- 导致的结果：疑罪从无
- 无罪推定是刑事诉讼举证责任和不强迫自证己罪的基础>>> [[法条/法条细则/中华人民共和国刑事诉讼法#^t120\|刑诉第 120 条]]
- 强调：法院必须经过正当程序审判才能做出有罪判决
	- 最初目的：法院垄断定罪权，废除免予起诉制度
