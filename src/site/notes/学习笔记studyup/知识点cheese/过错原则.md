# 民第1165条Ⅰ

</div>


**第一千一百六十五条**　行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任。
依照法律规定推定行为人有过错，其不能证明自己没有过错的，应当承担侵权责任。 

</div></div>
