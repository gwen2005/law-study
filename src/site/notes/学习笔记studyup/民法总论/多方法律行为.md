# 构成要件
1. 事实行为或官方行为
2. 意思表示：多方法律行为由数个意思表示构成，是数个意思表示的结合体

>例子：
>结婚（法律行为）：婚姻登记（官方行为）+结婚合意（意思表示）
>保管合同（法律行为）：交付（事实行为）+合意（意思表示）
