形成权：通过单方意思无须他人协助即可<font color="#d83931">改变特定法律状况</font>的权利 #背诵/概念 
# 特征
1. 权利性：其对自己可以保护一定权益
	- 排除非主观利益：代理权、监护权
2. 单方性：以形成权人单方的意思表示来行使，一经行使，目的即实现，形成权即消灭
3. 他决性：单方的意思表示就可以改变相对人的权利，相对人被形成权人单方面决定
	- 与支配权对比：
		- 形成权：决定权利的实现，但不会使他人处于被自己决定的状态
		- 支配权：可能使得他人处于被自己决定的状态
4. 不可侵性
>形成权系赋予权利人得依其意思而形成一定法律效果的法律之力，相对人并不负有相对应的义 务，只是受到拘束，须容忍此项形成权及其法律效果。形成权仅得由权利人行使，第三人无从加 以侵害，故非属“民法”第184条第1项前段所称的权利^[王泽鉴：《民法总则》，北京大学出版社2022年重排版，第113页]

# 内涵：改变法律状况
1. 直接使既存的法律关系发生变动
	- [[学习笔记studyup/民法总论/抵销权\|抵销权]]
	- [[学习笔记studyup/民法总论/减价权\|减价权]]
	- [[学习笔记studyup/民法总论/合同续期选择权\|合同续期选择权]]
	- 选择之债的选择权：其行使导致债务标的之变动>>> [[法条/法条细则/民法典/中华人民共和国民法典#^t515\|民第 515 条]]
2. 使一项法律行为成立或者效力状况发生变化，进而间接导致法律关系变动
	- 继续性合同的解除权（终止权）：其行使导致合同效力终止>>>>租赁合同的解除权
	- 撤销权
	- 🗣️ [[学习笔记studyup/民法总论/追认权\|追认权]]
	- 买回权、卖回权、先买权：其行使导致权利人与相对人之间成立一项买卖合同
>追认权、买回权等形成权行使行为属于辅助法律行为
>- 这些为单方法律行为，与其所辅助的法律行为（买卖合同等主法律行为）在概念上相互独立
>- 不能说买回权、卖回权、先买权等形成权的行使直接产生买卖债权债务关系，而无须承认权利人与义务人之间成立买卖合同
>>在买卖债权债务关系产生之后，债务的履行、对价风险的承担、瑕疵担保责任、合同的解除等问题仍须适用《民法典》中的买卖合同规则以及合同编通则的规定
>>如果说此类形成权的行使直接产生债权债务关系，则该债权债务关系就成为单方允诺之债而非合同之债

## 特殊类别：名为请求权实为形成权
1. 共有物分割请求权
> [[法条/法条细则/民法典/中华人民共和国民法典#^t303\|民第 303 条S1]]：共有人约定不得分割共有的不动产或者动产，以维持共有关系的，应当按照约定，但是共有人有重大理由需要分割的，可以请求分割；没有约定或者约定不明确的，按份共有人可以随时请求分割，共同共有人在共有的基础丧失或者有重大理由需要分割时可以请求分割。
2. 减价请求权
> [[法条/法条细则/民法典/中华人民共和国民法典#^t582\|民第 582 条S2]]：对违约责任没有约定或者约定不明确，依据本法[[学习笔记studyup/民法总论/形成权#^t510\|第五百一十条]]的规定仍不能确定的，受损害方根据标的的性质以及损失的大小，可以合理选择请求对方承担修理、重作、更换、退货、减少价款或者报酬等违约责任。
3. 遗失物善意取得时，所有人的所有物返还请求权
> [[法条/法条细则/民法典/中华人民共和国民法典#^t312\|民第 312 条S2]]：该遗失物通过转让被他人占有的，权利人有权向无处分权人请求损害赔偿，或者自知道或者应当知道（善意）受让人之日起二年内向受让人请求返还原物；但是，受让人通过拍卖或者向具有经营资格的经营者购得该遗失物的，权利人请求返还原物时应当支付受让人所付的费用。

## 行使的意思表示：单方法律行为
### 原则上不得附条件、期限
例外：条件/期限不影响确定性
- 所附条件是否成就取决于相对人：没有让相对人处于不确定的状态
- 所附期限为确定期限
- 相对人同意权利人附条件、期限
### 不得撤销，但可撤回
- 任何一个意思表示，一律可以撤回
- 原则上不能撤销，除非有法律的特别规定
	- 特别：要约、消费者的无理由退货权
### 不发生效力待定
若有瑕疵，直接无效
- 如果效力待定，则无法起到终结不确定性的作用
# 移转
## 不独立形成权
不可单独让与形成权，但可代理、授权行使、传达
- 性质：形成权权仍然保留在权利人手中，没有发生移转
### 与法律关系结合：随法律关系整体而移转

**第五百五十五条**　当事人一方经对方同意，可以将自己在合同中的权利和义务一并转让给第三人。

【案情】甲、乙签订买卖合同，乙未依约付款，经甲催告后乙仍未付款，甲依享有解除权
- 若解除权单独转让于丙：丙运用此权很可能构成滥用，因此解除权不能单独转让于丙
	- 甲乙双方之债权债务有可能违背甲乙的意愿而解除
	- 解除甲、乙之间的买卖合同对于丙来说并没有为其带来利益
- 若解除权只随债权一并转让
	- 甲转让价金债权后仍负债务，而丙行使解除权则会妨碍甲债务的履行，可能违背甲之意愿
	- 丙也无法只有债权情况下通过解除合同来实现其利益：解除只会让丙之债权消灭
- 若解除权只随债务一并转让
	- 甲转让债务须经乙同意，乙同意债务由丙承担显然是相信丙有履行债务的意愿和能力，若丙有解除权，一经解除，丙之债务、乙之债权都消灭，那么乙之利益就会受到损害
- ==甲只能在债权债务全部移转于丙的情况下（概括移转），才能将该买卖合同的解除权移转于丙==
