# 判断标准
## 辨析：沉默是否构成欺诈
取决于沉默者是否有[[学习笔记studyup/民法总论/告知义务\|告知义务]]或者提醒义务
- 如果表意人只要尽到其自身利益所要求的注意即可获取这些信息，那么他就无权期待相对人予以告知
