# 民第 154 条

</div>


**第一百五十四条**　行为人与相对人恶意串通，损害他人合法权益的民事法律行为无效。 

</div></div>


- 可以理解为对[[法条/法条细则/中华人民共和国民法典#^t153\|民第 153 条Ⅱ]]的特殊规定，适用154时须兼顾153Ⅱ的要件
