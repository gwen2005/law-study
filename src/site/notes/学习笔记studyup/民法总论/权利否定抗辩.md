# 特点
1. 由法官依职权主动适用
2. 即便被请求人（被告）没有明确主张抗辩，只要被请求人或者请求人（原告）在诉讼过程中提到与该抗辩相关的事实，法官就应依证明责任规则对该事实予以审查
3. 抗辩成立的，法官应径行适用
