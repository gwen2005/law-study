# 基本内容
对构成要件的解释以法条的保护法益为指导，而不仅停留在法条的字面含义上
- 解释对象：犯罪的构成要件而非案件事实
	- 明确该犯罪的保护法益
	- 在刑法用语可能具有的含义内确定构成要件的具体内容
