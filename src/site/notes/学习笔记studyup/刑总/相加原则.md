# 内容
根据有罪必罚和一罪一罚原则，认为数罪合并的刑罚，就是各罪刑罚的综合，也就是对数罪分别宣告刑罚，然后数刑相加，合并执行
# 特点
- 优势：使各具体犯罪都得到实际的惩治，体现了朴素的公平观念
- 劣势：会导致刑罚无限加重，在司法实践中很难操作
