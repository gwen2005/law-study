# 适用情况
当某种行为并不处于刑法用语的核心含义之内，但具有处罚的必要性与合理性时：对刑法用语作扩大解释
- 原则：反对专业解释，除非该解释有利于行为人
# 刑法表现
审判时怀孕的女人---执行前怀孕的女人
